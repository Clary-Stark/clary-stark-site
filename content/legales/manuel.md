steps:
  - num: "1"
    title: Connexion du disque dur
    body: Branchez le SSD Memoire sur un port USB-C de votre ordinateur, tablette ou smartphone compatible. Utilisez le cable USB-C fourni dans le coffret.
  - num: "2"
    title: Lancement du logiciel
    body: "Sur Windows/macOS : ouvrez le lecteur Memoire dans votre explorateur de fichiers et lancez Memoire.exe (Windows) ou Memoire.app (macOS). Sur mobile Android/iOS USB-C : l'application s'ouvre automatiquement."
  - num: "3"
    title: Creation de votre mot de passe
    body: Au premier lancement, creez un mot de passe maitre securise. Ce mot de passe chiffre l'integralite de vos donnees avec l'algorithme AES-256. Conservez-le precieusement.
  - num: "4"
    title: Ajout de vos informations
    body: "Utilisez les modules dedies pour renseigner : photos et videos, messages personnels, documents importants, informations confidentielles, biographie, carnet medical."
  - num: "5"
    title: Designation des beneficiaires
    body: Dans les parametres, definissez la ou les personnes de confiance qui auront acces a vos donnees. Un code d'acces secondaire leur sera reserve.
  - num: "6"
    title: Sauvegarde et mise a jour
    body: Il est recommande de brancher regulierement le disque pour enrichir vos informations. Les mises a jour du logiciel sont gratuites a vie et disponibles sur clary-stark.com.
